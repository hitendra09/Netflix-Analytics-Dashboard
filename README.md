# 🎬 Netflix Content Analysis Dashboard
## 📌 Project Overview

This project presents an interactive Netflix Content Analysis Dashboard developed in Power BI. The dataset was cleaned, transformed, and analyzed to generate meaningful insights about Netflix's content library.

The dashboard helps users explore trends related to movies, TV shows, genres, ratings, release years, countries, and content growth over time.

## 📊 Dashboard Features
Total Titles, Movies, TV Shows, Countries, and Latest Release Year KPIs
Top Countries by Content Distribution
Content Released by Year Analysis
Movie Duration Analysis
Movies vs TV Shows Comparison
Content Added Over Time
Top Genres Analysis
Content Rating Distribution
Interactive Slicers for:
Type
Country
Release Year
Rating
Genre
Date Range
## 🛠️ Tools & Technologies
Power BI
Power Query
DAX
Microsoft Excel / CSV Dataset
## 🔄 Data Preparation

The dataset was processed using Power Query with the following steps:

Removed duplicate records
Handled missing/null values
Corrected data types
Cleaned text columns
Created calculated columns and measures
Optimized data model for reporting
## 📈 Key Insights
Netflix content is dominated by Movies (69%) compared to TV Shows (31%).
The United States contributes the highest number of titles.
Drama and Comedy are among the most popular genres.
TV-MA is the most common content rating.
Content additions increased significantly after 2015.
## 📂 Dataset

Source: Netflix Titles Dataset (CSV)
